Ce projet permet de creer des alias plus courtes pour les commandes usuelles de l'outil GIT.

# Pour copier le repo:

´´´git clone https://github.com/riantsoa-randria/fancy_tools.git
´´´

# aller dans le repo:

´´´cd fancy_tools
´´´


# pour l'installation:

´´´./install.sh
´´´

# aliases:
´´´  gss='git status'
     gap='git add .'
     gpl='git pull'
     gpu='git push'
´´´
